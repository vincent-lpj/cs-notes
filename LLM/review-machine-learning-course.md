#### 77. Why Machine Learning?

In this section, we discuss about why we use ML to solve problems, and why it can not solve all the problems.

###### Overview of ML

ML is a **subset** of Artificial Intelligence, in general is the study of statistical computer algorithms that improve **automatically** through **data**.

- It means that unlike typical computer algorithms that rely on human input for whatever approch to take
- ML infer best approach from data itself
- Because effective ML depends largely on good data, we will spend a lot of time **cleaning** and **organizing data**.
- ML rely on a set of statistical methods that learns what **features** are important in data
- The structure of ML problem framing includes
  - Given features from a data set obtain a desired **label**
  - ML algorithms are often called **estimators** since they are estimating the desired label or output

###### What Problem Can ML Solve?

Typical problems will include:

1. Credit Scoring
2. Insurance Risk
3. Price Forecasting
4. Spam Filtering
5. Customer Segmentation
6. ...

For example, we can predict a price at which a house should sell at given features.

#### 78. Types of Machine Learning Algorithms

In the lessons later on, we will learn supervised learning first, and then unsupervised learning.

###### Supervised Learning

`Supervised Learning` uses **historical** and **labeled** data. The machine learning models predicts a value.

- Historical: known results and data from the past
- Labeled: desired output is known
  - Two main label types: **categorical** value for classification task, and **continuous** value for regression task

`Unsupervised Learning` applies to **unlabeled** data. The machine learning models discover possible patterns in the data.

- Group and interpret data without a label
- Major downside because there was no historical "correct" label
- Harder to evaluate the performance
