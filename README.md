## 💡 Commit Message

Format:

```
<type>: <short summary>
```

Types:

- notes: learning notes and summaries
- fix: typos or incorrect content
- docs: documentation changes
- refactor: reorganize without new content
- chore: miscellaneous tasks

Examples:

```
notes: add JavaScript fundamentals part 1
notes: summarize functions and arrays
fix: correct typos in section 2
docs: update README
refactor: reorganize section 3 notes
```
